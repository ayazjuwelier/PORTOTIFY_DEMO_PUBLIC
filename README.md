# Portotify – Demo Materials (Public)

This repository contains public demo assets for Portotify.

Included:

- **demo/** → demo scripts and narrative  
- **examples/** → example request payloads  
- **evidence/** → recorded outputs and governance responses  

⚠️ This repository intentionally **does not include the Portotify source code**.

The core system (engine, backend, governance layers) is maintained in a **separate private development repository**.

These materials are provided only to demonstrate how Portotify evaluates and governs AI decisions.

## How to explore the demo

1. Check `examples/` for example request payloads that can be sent to the Portotify API.
2. Review `evidence/` to see recorded outputs and governance decisions generated by the system.
3. Follow the narrative and explanation in `demo/` for the 3-minute demo flow.

These files illustrate how Portotify evaluates inputs, detects risk, and applies governance controls before or during AI execution.
